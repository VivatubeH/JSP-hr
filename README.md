<h2>자바 소스코드</h2>
<ul>
  <li>dao 패키지 - 데이터베이스 엑세스를 담당하는 자바 클래스들의 모임</li>
  <li>vo 패키지 - 데이터베이스의 객체들을 저장할 자바 클래스의 모임</li>
  <li>utils 패키지 - 자주 사용하는 기능을 구현해놓은 자바 클래스의 모임</li>
</ul>
